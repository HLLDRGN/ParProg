## Задание

Написать программу на C++ для перемножения двух квадратных матриц.

- **Входные данные:** файлы `matrix_a` и `matrix_b` с целочисленными матрицами
- **Выходные данные:** файл `result_matrix`, время выполнения, количество операций (2N³)
- **Верификация:** автоматическая проверка через Python / NumPy

## Реализация

Алгоритм умножения оптимизирован через предварительное **транспонирование матрицы B** — это обеспечивает последовательный (cache-friendly) доступ к памяти и снижает количество кэш-промахов.

| Функция | Описание |
|---|---|
| `load_grid()` | Чтение матрицы из текстового файла |
| `save_grid()` | Запись результирующей матрицы в файл |
| `transpose()` | Транспонирование для cache-friendly доступа |
| `multiply()` | Умножение через транспонированную B |

| N | N³ | Время (мс) | Время / N³ (×10⁻⁶) |
|:---:|---:|---:|---:|

Коэффициент стабилизируется в диапазоне **0.15–0.32 × 10⁻⁶**, что подтверждает кубическую сложность **O(N³)**.

Оптимизация через транспонирование дала ускорение ~**90×** по сравнению с наивным алгоритмом (400×400: 924 мс → 11 мс).
